# Cylc Server API Caller

## First see main README.md

## This App

- Displays raw suite JSON data for one suite.
- Vanilla Javascript

For the moment:
- digest authentication must be disabled in the suite server program
- self-signed SSL certificates must be ignored.

## Install
```
npm install
```

## Run (native)
```
npm start
```

## Run (browser)
```
firefox index.html
```
